# adidas-sales-dashboard-tableau
Interactive Tableau dashboard analyzing Adidas sales data to uncover business insights across regions, retailers, product categories, and sales channels. Features KPI tracking, profit analysis, sales trends, and data-driven storytelling using Tableau and Excel.

Adidas Sales Analytics Dashboard:

This project presents an interactive Tableau dashboard built using Adidas sales data. The dashboard provides a comprehensive overview of key business metrics, including sales performance, operating profit, operating margin, units sold, retailer performance, and regional trends.

Key Features
Sales & Profit KPI Tracking
Regional Performance Analysis
Retailer Comparison
Product Category Insights
Sales Method Analysis
Interactive Filters and Visualizations
Tools Used
Tableau
Microsoft Excel
Data Visualization
Business Intelligence
Objective

To transform raw sales data into actionable insights and demonstrate data storytelling, dashboard design, and analytical thinking using Tableau.
